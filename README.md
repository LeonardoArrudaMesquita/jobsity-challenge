## üì¶ Running Locally

First, Install the node_modules

```bash
$ npm install
```

Or

```bash
$ yarn
```

Create the environment variables file (.env) and fill them

```bash
$ cp .env.example .env
```

```
REACT_APP_ACCUWEATHER_API_KEY=
```

Then,

```bash
$ npm start
```

Or

```bash
$ yarn start
```

Finally, open your browser and visit http://localhost:8080/
